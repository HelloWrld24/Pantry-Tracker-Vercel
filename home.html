What is this? ‚ùì

This Pantry Tracker is a project to show off my newly learned skills in front-end and a bit of back-end programming. The project is connected with Firebase's Realtime Database, so any changes you make will be done in the database and reflected on the table that displays live data from the database. Feel free to add, remove, or delete items. Thanks for checking out my project!
